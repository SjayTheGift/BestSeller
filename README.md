
# Best Seller

Challange app


## Tech Stack

**Client:** Vannila Js, TailwindCSS

**Server:** Node

## Run Locally

Clone the project

```bash
  git clone https://github.com/SjayTheGift/BestSeller
```

Go to the project directory

```bash
  cd BestSeller
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
